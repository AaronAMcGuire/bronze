### Usage 
Bronze is easy to use and can be installed by adding a link to your sites HTML head element.
```
<link rel="stylesheet" href="https://dl.dropboxusercontent.com/s/fywgo105roorqgz/bronze-framwork.css?dl=0" />
```

### Features
- Tables 
- Fonts
- Nav Bars
- BG Colours
and more!

### Table Usage
```
<table>
  <tr>
   <th>Name</th>
   <th>Subject</th>
   <th>Score</th>
  </tr>
  <tr>
   <td>John</td>
   <td>Maths</td>
   <td>90%</td>
  </tr>
  <tr>
   <td>Jack</td>
   <td>Science</td>
   <td>87%</td>
  </tr>
  <tr>
   <td>Joe</td>
   <td>ICT</td>
   <td>98%</td>
  </tr>
</table>
```
